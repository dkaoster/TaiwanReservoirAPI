# TaiwanReservoirAPI

**台灣水庫資訊 API**

# How to use

`npx run TaiwanReservoirAPI`

## Clone

**Open your terminal**

`git clone git@github.com:dkaoster/TaiwanReservoirAPI.git`

**And then go to TaiwanReservoirAPI folder**

`cd TaiwanReservoirAPI/`

## Install

**Install Node modules**

`npm install`

## start

**Start this api**

`npm run start`

## See result

**Open your browser and type**

`http://127.0.0.1:9090/`
